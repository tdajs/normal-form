A (non-zero) integer matrix to be reduced to normal form. The matrix A can be of any size, as long as it is a well-formed non-empty matrix. For example,

[[1,2,3],[2,-5,0]]

is a valid input matrix of size 2Ã—3; whereas,

[[1,2,3],[2,-5]]
